4. state_manager.py (ANTI-NOISE)

Problem:
Gesture flickers = chaos

Solution:

if gesture == last_gesture for N frames:
    trigger()

Basic debounce:

5–10 consistent frames before action
5. action_dispatcher.py

This is where magic hits OS.

import subprocess

def execute(action):
    subprocess.run(action)

Mapping:

GESTURE_MAP = {
    "FIST": ["swaymsg", "workspace", "1"],
    "PALM": ["swaymsg", "workspace", "2"],
    "TWO": ["fish", "-c", "notify-send 'Yo'"]
}
6. main.py
while True:
    frame = get_frame()
    landmarks = detect_hand(frame)
    gesture = classify(landmarks)

    if stable(gesture):
        execute(GESTURE_MAP[gesture])
